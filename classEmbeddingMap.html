Detailed Description

template<size_t DimOfDomain, size_t DimOfRange>
class EmbeddingMap< DimOfDomain, DimOfRange >

Base class representing a coordinate mapping from a DimOfDomain-dimensional space (coordinates \(\xi^I\)) to a DimOfRange-dimensional space (coordinates \(x^i\)).
