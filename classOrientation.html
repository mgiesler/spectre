Detailed Description

template<size_t VolumeDim>
class Orientation< VolumeDim >

Allows one to construct a custom Orientation between two Blocks.

Usage

The user provides 2*VolumeDim Directions, encoding the correspondence between the directions in each block.

Template Parameters
VolumeDimthe dimension of the blocks.
